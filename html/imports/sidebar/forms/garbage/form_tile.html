Activate a tile

Before a tile can be used or referenced through our API, it needs to be activated.

Specify a tilename for the tile and it can be called through our API using @tilename in the same way you would Tweet to someone. If a name is not set, a random id will be assigned to the tile. Drag the sides of the polygon on the map to set the tile's area.
